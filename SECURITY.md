# Security Policy

## Supported Versions

Only the latest major release of `pkg-sentinel` receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of `pkg-sentinel` and the software supply chain seriously. If you discover a vulnerability in this tool, please do **not** file a public GitHub issue. Instead, report it privately:

- **Email**: security@pkg-sentinel.dev (PGP key available upon request)
- **Response time**: You can expect a response within 48 hours.

Please include:
1. Steps to reproduce the issue.
2. A description of the impact.
3. Any suggested remediation steps.

We will coordinate a patch and release a security advisory in compliance with responsible disclosure guidelines.
